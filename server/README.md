# PREQUISITES
    - make sure to have python 3.12.4 installed as this is the recommended python version for this project
    - this project uses uv as package manager instead of pip 

# SETUP GUIDE
    - pip install uv
    - create virtual env using `uv venv .venv --python 3.12`
    - activate virtual environment using `.venv\Scripts\activate`
    - install required packages and moduels using ``
